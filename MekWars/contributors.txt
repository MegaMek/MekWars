 ------
 MekWars makes extensive use of MegaMek (original author: Ben Mazur)
 code and mechanisms (Battle Value calculations, unit loading, actual
 gameplay and in innumerable other ways).
 ------
  Additonal MegaMek information may be found in the "MegaMek Docs" directory
  of a MekWars Client package. Project details and source may be found at:
  http://www.sourceforge.net/projects/megamek
  
 ------
 MekWars uses code derived from NFCChat to send messages between
 the client and server.
 ------
  NFCChat: http://sourceforge.net/projects/nfcchat
 
 ------
 MekWars uses Tom Gibara's TinyXML to
 parse data and configuration files.
 ------
 Tiny XML:
  Tom Gibara (original URL: http://www.grinninglizard.com/tinyxml/index.html)
